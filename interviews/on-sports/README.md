# On Interview Challenge Repository

## Overview

This repository exposes product information used by our website to sell products.

Currently, the `getProducts` function inside the `ProductsService` class returns our complete product catalog.

## The Challenge

As our products evolve, we will start launching new and improved versions for many of them. We want our customers to buy only the best (latest) version of each product.

**Your task:**
Update the code so that only the latest versions of products are returned.

## Useful Information

- The entry point is `index.ts`. Changes are watched automatically, and every time you save, the results of `ProductsService.getProducts` are printed. After your changes, this should return only the latest versions of products.
- There is no additional code, data, or information beyond what you see in this project.
- Do **not** use any external libraries; we expect a solution using only the provided code and TypeScript.
- Feel free to edit or add as many files as you consider necessary.

Good luck!
